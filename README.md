# TreeWeb
Веб-сервер для тестового задания. Операции CRUD, выдача каталогов в виде иерархии
Для запуска веб-сервера необходимо создать файл ".env" в корне, рядом с docker-compose.yml, который должен содержать текст:
JWT_SECRET_KEY="ваш ключ для шифрования"  
Ключ, к примеру, может быть таким - "mysuperpuper_secretsecretsecretkey!123"
